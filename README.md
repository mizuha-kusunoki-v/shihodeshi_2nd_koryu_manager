# 師匠弟子杯 進行管理アプリ

コラボ配信「師匠弟子杯」の当日進行を補助するWebアプリです。  
管理画面から試合を操作し、OBSのブラウザソースにリアルタイムで結果を反映します。

---

## 機能

- 参加者（師匠・弟子）の登録・削除
- Excelファイル（参加者リスト）からの一括インポート
- 弟子戦：出場回数が均等になるよう自動選出 ＋ 未経験の組み合わせを優先
- 師匠戦：5人以上登録時はランダムに4人を抽選して2v2
- OBSブラウザソース連携（1500msポーリング）
- 管理画面はBasic認証で保護

---

## 画面構成

| URL | 説明 |
|-----|------|
| `/admin` | 管理画面（配信者操作用・要認証） |
| `/obs` | OBS表示用画面（視聴者向け・1280×720） |

---

## ローカルでの起動

### 必要環境

- Python 3.11 以上

### セットアップ

```bash
cd deshihai
pip install -r requirements.txt
uvicorn main:app --port 8000
```

- 管理画面: http://localhost:8000/admin
- OBS表示: http://localhost:8000/obs

---

## Renderへのデプロイ

1. このリポジトリをフォーク
2. [Render](https://render.com) にGitHubアカウントでログイン
3. **New → Web Service** からフォークしたリポジトリを選択
4. `render.yaml` が自動で読み込まれるので **Deploy** を押すだけ

### 環境変数の設定

Renderの **Environment** メニューで以下を設定してください。

| Key | 説明 |
|-----|------|
| `ADMIN_USER` | 管理画面のユーザー名 |
| `ADMIN_PASS` | 管理画面のパスワード |

---

## Excelインポート形式

以下の列構成のExcelファイルに対応しています。

| 列 | 内容 |
|----|------|
| B列 | 弟子の名前 |
| C列 | 弟子の出欠（`〇` または `☆` で参加） |
| D列 | 師匠の名前 |
| E列 | 師匠の出欠（`〇` または `☆` で参加） |

## 技術スタック

- **バックエンド**: Python / FastAPI
- **フロント**: HTML / CSS / JavaScript（フレームワークなし）
- **データ保存**: state.json（サーバー再起動でリセット）
- **ホスティング**: Render（無料プラン）
